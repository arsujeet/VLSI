#SYNCHRNOUS 8-BIT UP COUNTER<br>
1.The input pins are Enable,Reset,Clock.<br>
2.Output we have a 8-bit up counter.<br>
3.We used always_ff "(posedge clk) begin" to behave the counter in a synchronised manner.<br>
4.Test Bench is written to test the System Verilog modules,by using many other testcases with delay mechanisms.<br>
